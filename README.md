# Nuestra historia

Proyecto web estático en **HTML, CSS y JavaScript vanilla** para crear una página emocional y visualmente delicada, pensada como regalo digital.

## Estructura
- `index.html`: estructura semántica de la single-page.
- `styles.css`: estilos visuales, responsive y animaciones.
- `script.js`: interacciones (scroll suave, carta final, reveal al hacer scroll).
- `assets/`: recursos para imágenes, música e íconos.

## Cómo abrir localmente
1. Clona o descarga este repositorio.
2. Abre `index.html` en tu navegador.

> También está listo para publicarse en **GitHub Pages**.
